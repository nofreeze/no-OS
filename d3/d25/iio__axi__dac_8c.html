char *  buf,

size_t  bytes_count,
uint32_t  ch_mask 
)

Write chunk of data into RAM. This function is probably called multiple times by libtinyiiod before a "iio_transfer_mem_to_dev" call, since we can only write "bytes_count" bytes at a time.

Parameters
iio_inst- Physical instance of a iio_axi_dac device.
buf- Values to write.
offset- Offset in memory after the nth chunk of data.
bytes_count- Number of bytes to write.
ch_mask- Opened channels mask.
Returns
bytes_count or negative value in case of error.
